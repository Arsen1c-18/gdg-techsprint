AI Study Helper

AI Study Helper is a lightweight web application that helps students understand topics quickly by generating clear and simple explanations using Google Gemini AI.

Overview

The application allows users to enter any topic and instantly receive an easy-to-understand explanation. It focuses on quick concept clarity with a clean, distraction-free interface.

Technology Used

React

Vite

Tailwind CSS

Google Gemini API

Deployment

The project is hosted on Vercel and uses environment variables to securely manage the Gemini API key.
